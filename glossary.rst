Glosar
======

.. glossary::

    Proiect
        Un *Proiect* reprezinta un folder compus dintr-o Aplicatie, un set
        de bundle-uri, librarii externe, un autoloader si scripturile
        controlorilor frontali.

    Aplicatie
        O *Aplicatie* reprezinta un folder ce contine *configuratia* pentru
        un set dat de Bundle-uri.

    Bundle
        Un *Bundle* consta intr-un set structurat de fisiere (fisiere PHP,
        stylesheet-uri, JavaScript-uri, imagini, ...) care *implementeaza* o
        singura facilitate (un blog, un forum, ...) si care poate fi usor
        impartasit cu alti dezvoltatori.
